#shell工具集

##脚本列表
* 安装jdk1.7: `curl https://coding.net/u/ethanzhu/p/shell/git/raw/master/jdk1.7-install.sh | sh`

* 配置加速docker镜像： `curl https://coding.net/u/ethanzhu/p/shell/git/raw/master/docker-new-setmirror.sh`
